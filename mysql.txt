流程控制语句 : 
	if 判断 语法  : 
		IF search_condition THEN statement_list 
		[ELSEIF search_condition THEN statement_list] ... 
		[ELSE statement_list] 
		END IF

	demo :

		IF age>20 THEN SET @count1=@count1+1;  
		ELSEIF age=20 THEN SET @count2=@count2+1;  
		ELSE SET @count3=@count3+1;  
		END IF;
