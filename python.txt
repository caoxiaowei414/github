   对象的结构
   	每个对象都要保存三种数据
   		id --> 用来标识对象的唯一性,每一个对象都有唯一的id  可以使用id() 函数查看
   				id由解析器生成,在cpython 就是对象的内存地址
   				兑现一但创建,他的id永远不会改变
   		type --> 类型   类型决定了对象有哪些功能
   		value --> 值  对象中存储的具体的值

   		对象分为 可变对象 不可变对象


   变量 和 对象

    对象并没有直接存储到对象中, 在python中变量更像是给对象取了一个别名
    变量中存储的是对象的 id

	python 标识符 (变量名 或者 方法名)
		以 (_foo)下划线的标识符 代表不能直接访问类的属性 , 需要通过类提供的接口进行访问  , 不能通过 from xxx import * 导入
		以 (__foo)双下划线的标识符 代表类的私有成员  
		以 (__foo__)双下划线开头和结尾 , 是 python 中特殊方法的专有标识 , 如 __init__ 代表类的构造函数
